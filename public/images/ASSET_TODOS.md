# China 1 Non-Food Image Assets

These non-food image assets are used by the site UI:

- `/images/panda-logo.png` - panda logo or mascot.
- `/images/bamboo-forest-top.jpg` - bright bamboo canopy/forest hero background.
- `/images/bamboo-forest-mid.jpg` - bamboo forest continuation for page body.
- `/images/bamboo-forest-bottom.jpg` - darker bamboo forest floor/footer background.
- `/images/bamboo-left.png` - transparent left-side bamboo stalk/leaf layer.
- `/images/bamboo-right.png` - transparent right-side bamboo stalk/leaf layer.

Do not add food photos to this directory for the website UI.
